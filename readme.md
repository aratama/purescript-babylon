
# purescript-babylon

Gates of Babylonjs (http://www.babylonjs.com/) for PureScript.

![screenshot](https://rawgit.com/aratama/purescript-babylon/master/docs/screenshot2.png)

Experimental. Far from complete APIs.

## build

```
$ psc --version
0.10.2
$ npm install
$ bower update
$ npm run build
```
